# Telco Customer Churn Analysis

## Objective
Analyze churn behavior and identify key drivers of customer attrition.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Key Findings
- Month-to-month contracts have the highest churn
- Fiber optic customers churn more
- High monthly charges correlate with churn

## Business Recommendations
- Promote long-term contracts
- Offer retention plans for fiber customers
- Re-evaluate pricing strategy for high-paying customers
